## 介绍

> 赠君一法决狐疑，不用钻龟与祝蓍。 试玉要烧三日满，辨材须待七年期。 
  周公恐惧流言日，王莽谦恭未篡时。 向使当初身便死，一生真伪复谁知？

道、法、术、器。有道无术，术可成；有术无道，止于术。道为术之灵，术为道之体；以道统术，以术得道。

求道之人，不问寒暑。

學習亦修行 ，且借朝隱， 且借紙遁，知足不足， 知止不止。小可修身，大可治国。

剥开被构建的概念，你我才是真正的人。
